# uC/SMTPc

µC/SMTPc is Micrium's SMTP (Simple Mail Transfer Protocol) client. It features:

* Delivered with complete 100% ANSI C source code and in-depth documentation.
* Easy configuration
* Simple API

µC/SMTPc is a component of µC/TCP-IP.
